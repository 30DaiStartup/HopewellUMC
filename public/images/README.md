# Image Upload Guide

Please save your uploaded church photos to this directory with the following filenames:

1. **church-exterior.jpg** - The exterior brick church building photo
2. **children-ministry.jpg** - Children sitting in sanctuary during ministry time
3. **volunteers-packing.jpg** - Volunteers in teal shirts packing boxes in sanctuary
4. **womens-breakfast.jpg** - Women smiling at breakfast table (first group photo)
5. **mens-group.jpg** - Men's group photo at breakfast
6. **woman-speaking.jpg** - Woman in white vest speaking at podium
7. **womens-group.jpg** - Second women's breakfast group photo
8. **outdoor-work.jpg** - Volunteers doing outdoor construction/repair work

- **Carousel**: church-exterior.jpg, children-ministry.jpg, volunteers-packing.jpg
- **Home Page - Everyone is Welcome**: womens-breakfast.jpg
