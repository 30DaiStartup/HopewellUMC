## Prayer Request Submission Form

The page features an interactive form allowing community members to submit prayer requests.

### Form Fields

**Required**:
- Name
- Title/Subject
- Prayer request details
- Email address

**Optional**:
- Category selection
- Display preference (Public/Private)

### Categories Available

1. Health/Hospital
2. Praise
3. Grief
4. Travel
5. Salvation
6. Missions
7. Nursing Care
8. Military Personnel
9. Persecution
10. Other

### Privacy Options

- **Public**: Prayer request visible to all site visitors
- **Private**: Prayer request only visible to church staff/prayer team

## Features

1. **Real-time Updates**: Prayer requests display chronologically
2. **Category Filtering**: Requests organized by category
3. **Submitter Information**: Each request shows who submitted it
4. **Date Tracking**: Requests include submission dates
5. **Integration**: Connects with church calendar and ministry pages
